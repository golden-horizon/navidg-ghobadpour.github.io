About Me

I began my technical journey with Cisco networking, building a strong foundation in routing, switching, subnetting, VLANs, and network security fundamentals. From there, I transitioned into cybersecurity, focusing on governance, risk, and compliance frameworks including ISO/IEC 27001 and the Essential Eight maturity model. I have practical experience conducting security assessments, identifying vulnerabilities, preparing audit documentation, and reporting security gaps with remediation strategies aligned to organizational standards. My background includes hands-on work with SIEM monitoring, log analysis, firewall configuration, VPN deployment, and secure infrastructure design. Beyond security operations, I expanded into Linux system administration, secure server configuration, and full-stack web development using HTML, CSS, JavaScript, Python, and modern frontend/backend libraries. I focus on writing clean, secure, and scalable code while understanding the infrastructure that supports it. I combine networking, security governance, and development knowledge to approach problems holistically from infrastructure to application layer. I’m here to contribute to meaningful projects where security, cloud, and development intersect. If your team is building something secure, scalable, and impactful ,I’d be excited to be part of it.
